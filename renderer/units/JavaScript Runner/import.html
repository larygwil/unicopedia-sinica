Built-in Functions
  • $.clear ();
    Clears the output field.
  • $.getpath (name);
    Gets a predefined directory path by name, among: 'desktop', 'documents', 'downloads', 'userData', etc.
  • $.notify (message[, callback]);
    Sends a desktop notification with the given message and, if defined, executes the callback function when the user clicks on the notification.
  • $.save (string, filename[, dirname]);
    Saves a string to a text file on the desktop folder, and returns the saved file path (or null on error).
  • $.stringify (value[, replacer[, space]]);
    Converts a JavaScript value to a JSON string (with balanced spacing).
  • $.write (text[, text...]...);
    Writes the specified text to the output field.
  • $.writeln (text[, text...]...);
    Writes the specified text to the output field and appends a line break.
